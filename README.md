# Wellnest Site

This is your starter website package for Wellnest.

## What is included
- A working React/Vite website
- Product finder
- Clean swap finder
- Ingredient decoder mockup
- Registry sections
- Blog/SEO guide sections
- Affiliate disclosure area
- Editable product data in `src/App.jsx`

## How to preview it locally
1. Install Node.js from https://nodejs.org
2. Unzip this folder
3. Open the folder in VS Code
4. Open a terminal in the folder
5. Run:
   npm install
6. Run:
   npm run dev
7. Open the local URL it gives you

## How to edit products
Open `src/App.jsx`.

Find `const PRODUCTS = [...]`.

For each product, update:
- title
- category
- price
- rating
- tag
- retailer
- affiliateUrl
- badges
- why
- concerns

Replace affiliateUrl: "#"
with your real affiliate link.

## Suggested launch path
1. Create affiliate accounts
2. Replace placeholder products with real products
3. Upload to GitHub
4. Connect GitHub to Vercel
5. Buy/connect a domain
6. Start Pinterest and TikTok content
7. Add Google Analytics or Plausible
